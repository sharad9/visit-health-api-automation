# GWT API Flow Builder

Minimal GWT project with a single EntryPoint, structured to mirror a Vaadin-style layout.

## Build

```bash
cd /Users/sharadrakeshmaddheshiya/Desktop/vanilla/gwt-api-flow-builder
mvn -q -DskipTests clean package
```

Output will be generated into:
```
target/gwt-api-flow-builder-1.0.0
```
